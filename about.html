About:

This is a simple website that lists all of the Pokémon games I own. I have a lot of Pokémon games. This website dynamically displays information about each game: an image of the game, and the name of the game; and the background is color-coded to represent the status of the game in my collection.

Statement of Purpose:

I created this website to showcase my collection of Pokémon games. I wanted to create a website that was simple, easy to navigate, and visually appealing. I also wanted to create a website that was dynamic and responsive to user input.

As I continue expanding my collection, I will add them to this website. I also have plans to add a page for Pokémon spin-off games—the home page only list the mainline Pokémon games released for the handheld Nintendo systems—but my collection does not yet include enough spin-off games to showcase.

I hope you enjoy browsing my collection of Pokémon games!
